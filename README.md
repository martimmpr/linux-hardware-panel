# Hardware Panel

## Overview

Hardware Panel is a powerful system monitoring and hardware control application for Linux. Built with Python and PyQt5, it provides real-time hardware monitoring, fan control and dynamic power profile management, through an intuitive interface.

Compatible with all major Linux distributions including Fedora, Ubuntu, Debian, Arch Linux, and others.
